# React Projects — freeCodeCamp "Master React by Building 25 Projects"

The first 6 projects from the course, built with React + Vite + Tailwind CSS.
Use the sidebar in the app to switch between them.

1. Accordion
2. Random Color Generator
3. Star Rating
4. Image Slider
5. Load More Button
6. Tree View / Recursive Menu

## Run it

npm install
npm run dev

Then open the URL Vite prints (usually http://localhost:5173).

## Build for production

npm run build
npm run preview

## Project structure

src/
  components/     <- each project as its own component
  App.jsx         <- sidebar + switches between projects
  index.css       <- Tailwind directives

## Stack

- React 19
- Vite
- Tailwind CSS
- lucide-react (icons)
