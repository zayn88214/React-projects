# React Projects

A collection of beginner-friendly React projects built to practice modern React development.

## Projects

- 🍽️ Food Recipe App
- 🌦️ Weather App

## Getting Started

Clone the repository and install the dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit the local URL shown in the terminal (typically `http://localhost:5173`).
